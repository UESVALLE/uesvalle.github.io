# Portal de Tableros UESVALLE

Repositorio institucional para la publicación de tableros HTML, datos CSV, scripts de normalización y documentación técnica asociados a los procesos de seguimiento, vigilancia y control sanitario de UESVALLE.

## Estructura

- `index.html`: portal principal de acceso a tableros.
- `dashboards/`: tableros HTML por módulo.
- `data/`: datos operativos y procesados.
- `scripts/`: scripts de normalización y actualización.
- `docs/`: documentación técnica.
- `archive/`: respaldos y versiones anteriores.

## Módulos principales

- Seguimiento Actividades Agua para Consumo Humano
- IRCAS
- Filtros
- Dosificación de cloro
- Piscinas
- Empresas Aplicadoras de Plaguicidas
- Muestras
- Mapas de Riesgo MPR
